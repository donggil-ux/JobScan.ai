# JobScan.ai

> 5년차 이상 프로덕트 디자이너·UX 디자이너를 위한 AI 채용공고 큐레이션 사이트

**라이브 데모:** https://jobscan-ai.vercel.app

---

## 주요 기능

### 실시간 채용공고 수집
원티드, 리멤버, 사람인, 잡코리아 4개 플랫폼에서 공고를 병렬로 수집합니다.
API 실패 시 더미 데이터로 자동 폴백하며, 30분 캐싱으로 불필요한 반복 요청을 방지합니다.

### AI 매칭 스코어
65~95점 범위의 매칭 스코어를 산출해 공고마다 표시합니다.
점수에 따라 🔥 / ✨ / 💡 이모지와 색상으로 구분합니다.

### 멀티 필터 + 검색
경력(5~7년 / 8년+), 고용형태, 기업규모, 플랫폼 4가지 필터를 조합해 사용할 수 있고,
회사명·포지션명 실시간 검색도 지원합니다.

### 기업 심층 정보 상세 페이지
카드 클릭 시 `/company/[name]` 페이지로 이동합니다.
TheVC · 혁신의숲에서 기업의 비전, 설립 연혁, 매출액, 누적 투자금, 임직원 수를 자동 수집합니다.
하단 고정 CTA 버튼을 통해서만 원본 채용 공고로 이동합니다.

### 회사 재무 정보 (옵션)
DART(금감원 공시) + 국민연금 API를 통해 매출액·직원수를 카드에서 바로 확인할 수 있습니다.
(환경 변수 설정 필요)

### 스와이프 인터랙션
모바일에서 카드를 오른쪽으로 스와이프하면 📌 스크랩, 왼쪽으로 스와이프하면 숨기기 처리됩니다.

## 시작하기

```bash
# 의존성 설치
bun install

# 개발 서버 실행 (http://localhost:3001)
bun dev

# 프로덕션 빌드
bun run build
```

### 환경 변수 (선택)

`.env.local` 파일을 생성하고 아래 값을 설정하면 회사 재무 정보가 활성화됩니다.

```env
DART_API_KEY=your_dart_api_key      # 금감원 DART — 매출액 조회
NPS_API_KEY=your_nps_api_key        # 국민연금 — 직원수 조회
```

API 키 없이도 사이트는 정상 동작합니다.

## 프로젝트 구조

```
src/
├── app/
│   ├── page.tsx                      # 메인 피드 (필터·검색 상태 관리)
│   ├── layout.tsx                    # 앱 레이아웃
│   ├── globals.css                   # 전역 스타일 (Pretendard 폰트)
│   ├── company/
│   │   └── [name]/
│   │       └── page.tsx              # 기업 심층 정보 상세 페이지
│   └── api/
│       ├── jobs/route.ts             # 채용공고 수집 API (4개 플랫폼)
│       ├── startup-info/route.ts     # 기업 정보 스크래핑 API (TheVC · 혁신의숲)
│       └── company-info/route.ts    # 재무 정보 API (DART · 국민연금)
├── components/
│   ├── Header.tsx                    # 상단 타이틀
│   ├── FilterBar.tsx                 # 검색 + 필터 칩 (스티키)
│   ├── FilterChip.tsx                # 개별 필터 드롭다운
│   ├── JobFeed.tsx                   # 공고 목록 + 스크랩 카운트
│   └── JobCard.tsx                   # 공고 카드 (스와이프 · 회사정보)
├── data/
│   └── jobs.ts                       # 폴백용 더미 데이터
└── types/
    └── job.ts                        # Job, Filters 타입 정의
```

## API 엔드포인트

| 엔드포인트 | 설명 | 캐싱 |
|------------|------|------|
| `GET /api/jobs` | 4개 플랫폼 채용공고 수집 | 30분 |
| `GET /api/startup-info?companyName=토스` | 기업 비전·투자·매출 스크래핑 | 없음 |
| `GET /api/company-info?name=토스` | DART 매출 + 국민연금 직원수 | 24시간 |

## 비관련 공고 필터링

프로덕트/UX 디자이너와 무관한 직군(그래픽, 영상, 패션, 개발자 등)을 자동으로 제거합니다.
헤드헌팅·인력파견 회사(프로써치, 맨파워 등)도 전 플랫폼에서 제외됩니다.
